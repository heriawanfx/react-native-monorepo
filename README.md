## Installation
```
git clone https://github.com/heriawanfx/react-native-monorepo.git <project_folder>
cd <project_folder>
yarn
```
To start WebApp
```
yarn web
```
To start Native android app
```
yarn android
```
To start Native ios App
```
yarn ios
